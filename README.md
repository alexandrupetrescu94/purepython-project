purepython-project
==================

Course Project for http://purepython.eaudeweb.ro

System prerequisites
--------------------

1. Install pip:
  
  ```
  wget https://bootstrap.pypa.io/get-pip.py
  sudo python get-pip.py
  ```

2. Install Django 1.6: 

  ```
  sudo pip install Django==1.6.2
  ```
